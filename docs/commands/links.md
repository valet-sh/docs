# Links


## Description

Output all links in a list that have been created with *[link](/commands/link)*.

## Usage

``` bash
valet.sh links
```

## Example Output

``` bash
+----------------+----------------------------------------+------+-------------+-------------+
|      Name      |                  Path                  | SSL  | PHP-Version | Application |
+----------------+----------------------------------------+------+-------------+-------------+
|    projectx    |   /home/user/workspace/projectx        | True |    php73    |  Magento 2  |
+----------------+----------------------------------------+------+-------------+-------------+
```
