    MailHog

    Quote

    "Mailhog is an email testing tool for developers" - please visit https://github.com/mailhog/MailHog for more information.

    Usage

    You can access the UI via https://mailhog.test

    PHP is configured to use mailhog automatically (e.g. the mail() function), but you can use SMTP with host localhost and port 8025 as well.
